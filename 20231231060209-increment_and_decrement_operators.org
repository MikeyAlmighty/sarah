* Increment and Decrement

=Increment= and =Decrement= operators, ~++~ and ~-­~,respectively, can be applied to [[id:2483364b-73d3-4169-92b9-1fea473a9b01][numeric vriables]] and have a *high order of precedence* compared to [[id:9bcf2aef-060a-4011-98b5-cb949a9f7d1a][Binary Operators]]

#+ATTR_ORG: :width 600
[[./assets/increment_and_decrement_operators.png]]

** Increment ~++~

#+BEGIN_SRC java
int parkAttendance = 0;
System.out.println(parkAttendance); // 0
System.out.println(++parkAttendance); // 1
#+END_SRC

** Decrement ~--~

#+BEGIN_SRC java
int parkAttendance = 1;
System.out.println(parkAttendance); // 1
System.out.println(parkAttendance-­); // 1
System.out.println(parkAttendance); // 0
#+END_SRC

** _Exam Trick_

Know the difference between ~pre(--x)~ and ~post(x--)~ =Increment/Decrement=.
