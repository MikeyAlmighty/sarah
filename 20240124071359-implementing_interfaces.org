* Implementing Interfaces

An *interface* is an ~abstract data~ type that declares a list of [[id:3e6ed9f7-a719-4f68-94f0-29cb4914ebf2][abstract methods]] that any class implementing the interface *must* provide.

Over time, the precise definition of an interface has changed, as new method types are now supported.
