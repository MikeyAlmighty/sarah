#+title: Cryptanalysis Attacks

There are several types of attacks that [[id:2641c0d8-e9ac-47cf-a3a9-6be59696ceb4][hackers]] or security professionals employ to get data from a network using [[id:29235b07-ca3d-485e-b733-bf223a30b80e][Cryptanalysis]].
The attacks all use a different way into the network to gain encoded information and translate it from the encoded form into information that can be easily read.
