#+title: Windows 10 vs Windows 11
#+filetags: :OS:

Windows 10 and Windows 11 primary difference is aesthetic.

* Features

| FEATURE         | Win10                                      | Win11                            |
|-----------------+--------------------------------------------+----------------------------------|
| Apps            | Added from Windows store or manual install | Android apps added natively      |
| Virtual Desktop | unintuitive                                | More user friendly               |
| Teams           | Included (defaults to Skype)               | Featured (and Default)           |
| Widgets         | xxxx                                       | accessed from taskbar directly   |
| Touch and Pen   | xxxx                                       | More support (Vibration for pen) |
| RAM             | Lowest: 1GB 32-bit, 2GB 64-bit             | Base 4GB                         |
