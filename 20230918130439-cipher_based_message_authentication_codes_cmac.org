#+title: Cipher-Based Message Authentication Codes (CMAC)

Cipher-based [[id:5b075cfc-c900-4457-8811-27152afd5245][MAC]]

The process is similar to [[id:81b25db9-3b85-4542-aa73-f6d76cabc6c0][HMAC]], but instead of using a [[id:48e1ad8d-51d5-4656-b292-8f2ba311f061][hashing]] function to produce a digest, a symmetric [[id:ff735327-4e77-43b6-8755-50ceccc7f9a0][cipher]] with a shared key is used to encrypt the message and the resulting output is used as the [[id:5b075cfc-c900-4457-8811-27152afd5245][MAC]].
