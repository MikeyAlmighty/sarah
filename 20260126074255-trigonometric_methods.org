#+title: Trigonometric Methods

* Trigonometric Methods

| Method               | Description                                               | Example                                     |
| -------------------- | --------------------------------------------------------- | ------------------------------------------- |
| sin(radians)         | Returns the trigonometric sine of an angle in radians.    | `Math.sin(Math.PI / 2); // 1.0`             |
| cos(radians)         | Returns the trigonometric cosine of an angle in radians.  | `Math.cos(0); // 1.0`                       |
| tan(radians)         | Returns the trigonometric tangent of an angle in radians. | `Math.tan(Math.PI / 4); // 1.0`             |
| toRadians(degree)    | Converts degrees to radians.                              | `Math.toRadians(180); // 3.141592653589793` |
| toDegrees(radians)   | Converts radians to degrees.                              | `Math.toDegrees(Math.PI); // 180.0`         |
| asin(a)              | Returns the angle (radians) whose sine is `a`.            | `Math.asin(1); // 1.5707963267948966 (π/2)` |
| acos(a)              | Returns the angle (radians) whose cosine is `a`.          | `Math.acos(1); // 0.0`                      |
| atan(a)              | Returns the angle (radians) whose tangent is `a`.         | `Math.atan(1); // 0.7853981633974483 (π/4)` |
