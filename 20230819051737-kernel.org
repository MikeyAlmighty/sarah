#+title: Kernel
#+filetags: :OS:

The =kernel= is a computer program at the core of a computer's [[id:1dcff204-74cf-445d-9357-5c01ca3d2a03][OS]] and generally has complete control over everything in the system.

It is the portion of the [[id:1dcff204-74cf-445d-9357-5c01ca3d2a03][OS]] code that is always resident in memory and facilitates interactions between hardware and software components.
