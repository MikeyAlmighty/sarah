#+title: Group Policy Objects (GPOs)

Ways to manage the configuration of a [[id:45bb23df-e29d-46dc-b704-f17cf26af1d7][Windows]] machine
Set of policies and preferences that can be applied to a group of objects in the directory

* Configuration
Can contain:
- Computer Configuration (Computer starts)
- User Configuration (User logs on)

Policies and settings stored in [[id:6bd42d46-6a6a-456f-bac7-1879fa67ea0a][Windows Registry]]
