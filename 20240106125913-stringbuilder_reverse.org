* ~.reverse()~

It *reverses* the characters in the sequences and *returns a reference to the current* ~StringBuilder~.

** Signature

#+BEGIN_SRC java
public StringBuilder reverse()
#+END_SRC

*** Example

#+BEGIN_SRC java
var sb = new StringBuilder("ABC");
sb.reverse();
System.out.println(sb); // CBA
#+END_SRC
