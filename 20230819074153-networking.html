6. Casting

At the Network Layer, looking at a special bit in the MAC Address.

6.1. 1. Unicast

If the least significant bit in the first Octet of a destination address is set to 0, It means that Ethernet Frame is intended for only the destination address.

6.2. 2. Multicast

If the least significant bit in the first Octet of a destination address is set to 1, It means that Ethernet Frame is intended for a set of destination addresses.
