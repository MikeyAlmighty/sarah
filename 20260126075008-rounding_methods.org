#+title: Rounding Methods

* Rounding Methods

- [[id:80f9f7c5-8e8b-46e6-a3f7-345a8865e010][.floor()]]
- [[id:0dc14ab9-6e98-40fa-af07-2aa2ce4af1e8][.ceil()]]
- [[id:4670277c-8ea0-4e70-9815-d795bedea99b][.round()]]

| Method   | Description                                                        | Examples                |
|----------+--------------------------------------------------------------------+-------------------------|
| ceil(x)  | Rounds x up to the nearest integer (returned as a double).         | Math.ceil(4.2) → 5.0    |
|          |                                                                    | Math.ceil(-4.2) → -4.0  |
| floor(x) | Rounds x down to the nearest integer (returned as a double).       | Math.floor(4.8) → 4.0   |
|          |                                                                    | Math.floor(-4.2) → -5.0 |
| rint(x)  | Rounds to nearest integer; ties go to the even integer (double).   | Math.rint(2.5) → 2.0    |
|          |                                                                    | Math.rint(3.5) → 4.0    |
| round(x) | Rounds using floor(x + 0.5); returns int (float) or long (double). | Math.round(4.6f) → 5    |
|          |                                                                    | Math.round(4.5) → 5     |
