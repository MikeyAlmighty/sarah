Data Encryption Standard (DES)

Designed in the 1970s by IBM with some input form the NSA Adopted as a FIPS for the US

Symmetric Block Cipher that uses 64-bit key sizes and operates on blocks 64-bits in size

Though the key size is technically 64-bits in length, 8-bits are used only for Parity checking

Key Length = 56-bits (72 quadrillion possible keys)
