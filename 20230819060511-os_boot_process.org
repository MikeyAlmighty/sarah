#+title: OS Boot Process

#+ATTR_ORG: :width 900
[[./assets/Boot_Process.png]]

1. Computer powered on.
2. [[id:de18822a-c4d8-4094-90d6-359ec6b87e83][BIOS]]/[[id:0e0f66f9-1c8b-4604-b6a9-2d54d8509511][UEFI]] runs [[id:8e9f187f-93c3-486a-99ef-81e687e246d0][POST]].
3. Follow [[id:09ecc260-38eb-4402-975e-80d45bacfc5c][Boot Order]] to search for [[id:ba309c3d-889f-4d8e-87ed-85f9e1c12fd3][Boot Devices]]
4. Devices will be checked for a [[id:14503479-f553-46fa-83a9-c6e78ac080fc][Bootloader]]
5. Load [[id:14503479-f553-46fa-83a9-c6e78ac080fc][Bootloader]]
6. Load [[id:1dcff204-74cf-445d-9357-5c01ca3d2a03][OS]]
7. Load [[id:cba22c0a-759f-4b3a-9ee4-25663f8eb434][kernel]]
8. The [[id:cba22c0a-759f-4b3a-9ee4-25663f8eb434][kernel]] loads up drivers and more so that our hardware can talk to our software.
9. Essential system [[id:2bc74ef5-d9b1-4286-ab74-bc061c6a1704][processes]] and User space items are launched.
