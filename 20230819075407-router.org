#+title: Router

[[id:958904bc-6a62-45e5-a508-9f996ca87212][3. Network Layer]] Device

Connects lots of different devices together and helps route network traffic.

Knows how to forward data between independent networks.

Routers share data with each other via a protocol known as [[id:68b06da3-864a-4e54-8e1e-f48b1ca1e9f8][Border Gateway Protocol (BGP)]].

1. Inspect the contents of an IP datagram,
2. Decrement the [[id:4bf28699-146b-4eba-8d34-b3f58a20b68d][TTL]] by one
3. Recalculate the checksum
4. Forward the rest of the data at the network layer without touching it
