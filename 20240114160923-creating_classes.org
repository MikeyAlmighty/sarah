#+title: Creating Classes
#+property: header-args :tangle code/ocp/6/creating_classes.java

A =class= is a blueprint for creating *objects*, and *objects* are ~instances~ of =classes=.

This is a fundamental concept of [[id:f169a862-02f6-4ca9-a8b1-3de711c8ed10][Object Oriented Programming]].

* Creating Classes

#+BEGIN_SRC java
class Movie {
    // Fields
    String name;
    int releaseDate;

    // Constructor
    Movie(String name, int releaseDate){
        this.name = name;
        this.releaseDate = releaseDate;
    }

    // method
    void displayInfo(){
        System.out.println("Name: " + name + " , Release Date: " + releaseDate);
    }
}

public class ClassExample {
    public static void main(String[] args) {
        // Create an Object of the Class
        Movie movie = new Movie("Deadpool", 2016);
        movie.displayInfo();
    }
}
#+END_SRC
