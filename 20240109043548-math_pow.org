* .pow()

The ~pow()~ method handles exponents.

** Signature

#+BEGIN_SRC java
public static double pow(double number, double exponent)
#+END_SRC

*** Example

#+BEGIN_SRC java
double squared = Math.pow(5,2); // 5^2 = 25.0
#+END_SRC

**** _Exam Trick_

Note the result is =25.0= rather than =25= since it is a ~double~.
