#+title: Compiler

* Compiler

*Translates* the entire =source program= into a [[id:89afff2d-63e0-4cc8-87fb-b30d65018241][Machine Language]] file for *execution*.

Program that *translates* computer code written in one programming language (=source=) into another language (=target=).

Invented by [[id:f2498bfe-5984-4990-8282-5f7e748a4303][Admiral Grace Hopper]] in *1952*.
