#+title: File Metadata

Contains the information about our file.

- Who created it
- When it was last modified
- Permissions
  etc.

The file metadata tells us everything we need to know about our file.

It also tells us what type of file it is.

A =file extension= is the appended part of a filename that tells us what type of file it is in certain operating systems.
