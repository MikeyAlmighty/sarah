#+title: Git

*7 April 2005*

Distributed VCS that tracks changes in any set of computer files, usually used for coordinating work among programmers who are collaboratively developing source code during software development.

Its goals include speed, data integrity, and support for distributed, non-linear workflows (thousands of parallel branches running on different computers).
