* Chapter 4. Core APIs

** Exam Objectives

*** Handling ~date~, ~time~, ~text~, ~numeric~ and ~boolean~ values

- Use [[id:46582863-7f97-448d-bbc7-f7da4c1e7574][primitives]] and wrapper classes including Math API, parentheses, type promotion, and casting to evaluate arithmetic and boolean expressions.

- Manipulate ~text~, including ~text blocks~, using =String= and =StringBuilder= classes

- Manipulate ~date~, ~time~, ~duration~, ~period~, ~instant~ and ~time­zone~ objects using =Date­Time API=.

*** Working with Arrays and Collections
- Create Java ~arrays~, ~List~, ~Set~, ~Map~, and ~Deque~ *collections*, and =add, remove, update, retrieve and sort= their *elements*.
