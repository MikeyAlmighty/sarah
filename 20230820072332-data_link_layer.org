#+title: 2. Data Link Layer

Defining a common way of interpreting these signals so network devices can communicate.

Lots of protocols exist at the data link layer but the most common is known as [[id:514c3f0b-c3ec-46f1-aa28-c2a4874a6a7b][Ethernet]].

(Although wireless technologies are becoming more and more popular).
