* Java Primitive Data Types

| Type    | Size (bits) | Min Value                  | Max Value                 | Default   | Description                      |
|---------+-------------+----------------------------+---------------------------+-----------+----------------------------------|
| byte    |           8 | -128                       | 127                       | 0         | 8-bit signed integer             |
| short   |          16 | -32,768                    | 32,767                    | 0         | 16-bit signed integer            |
| int     |          32 | -2,147,483,648             | 2,147,483,647             | 0         | 32-bit signed integer            |
| long    |          64 | -9,223,372,036,854,775,808 | 9,223,372,036,854,775,807 | 0L        | 64-bit signed integer            |
| float   |          32 | ~1.4E-45                   | ~3.4E38                   | 0.0f      | 32-bit floating point (IEEE 754) |
| double  |          64 | ~4.9E-324                  | ~1.8E308                  | 0.0d      | 64-bit floating point (IEEE 754) |
| char    |          16 | 0                          | 65,535                    | '\\u0000' | Unsigned UTF-16 character        |
| boolean | JVM-defined | false                      | true                      | false     | Logical true/false               |

** Note
Java uses the ~32-bit~ =IEEE 754= for the ~float~ type and the ~64-bit~ =IEEE 754= for the ~double~ type.
