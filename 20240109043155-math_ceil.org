* .ceil()

The ~ceil()~ method takes a ~double~ value.

If it is a whole number, it returns the same value.

If it has any fractional value, it rounds up to the next whole number.

** Signature

#+BEGIN_SRC java
public static double ceil(double num)
#+END_SRC

*** Example

#+BEGIN_SRC java
double c = Math.ceil(3.14); // 4.0
#+END_SRC
