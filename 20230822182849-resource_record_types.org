#+title: Resource Record Types

* A Record
Used to point to a certain domain name at a certain [[id:1e255b7a-aac7-4c36-9bb1-4addf887e312][IPv4 Address]].

* Round Robin
Concept that involves iterating over a list of items one by one in an orderly fashion

* Quad A Record
Similiar to an =A Record=, except returns an [[id:1011a8fb-ad00-40bb-a1cc-4b53c9fedff0][IPV6 Address]]

* CName (Canonical Name) Record
Used to redirect traffic from one domain to another

* Mail Exchange (MX)
Used to deliver email to correct server

* Service Record (SRV)
Used to define location of various specific services.

* Text Record (txt)
Originally intended to be used only for associating some descriptive text with a domain name for human consumption.
