#+title: Binary Operators

* Binary Operators
Operators that take *two* operands

A [[id:6fab7cdd-d658-4d28-a1b8-492fc3943dac][unary Operator]] has only *one* operand (*binary* *operator* has two).

For example, the =–= operator in ~–5~ is a [[id:6fab7cdd-d658-4d28-a1b8-492fc3943dac][unary operator]] to negate number ~5~, whereas the =–= operator in ~4 – 5~ is a *binary* *operator* for *subtracting* ~5~ from ~4~.


Used to:
- Perform *mathematical* *operations* on [[id:ace39873-193a-42c3-a97d-0a25e630687e][variables]]
- Create logical expressions
- Perform basic variable assignments.
