Every class that implements the Collection Interface can return a stream

Mapping Filtering Slicing Sorting Stream Getting Unique elements Reducers

A call to .forEach is a terminal operation VS One can use .peek at any point to "peek" at the stream without termintating 

public static void show() {
        var movieStream = Stream.of(
                new Movie("b", 15),
                new Movie("a", 5),
                new Movie("d", 15),
                new Movie("e", 35),
                new Movie("c", 5)
            );
            movieStream
            .peek(movie -> System.out.println("entry: " + movie.getTitle()))
            .filter(movie -> movie.getLikes() > 10)
            .peek(movie -> System.out.println("filter: " + movie.getTitle()))
            .map(Movie::getTitle)
            .peek(movieTitle -> System.out.println("map: " + movieTitle))
            .forEach(System.out::println);
}

1. Primitive Streams

public static void show() {
    IntStream.range(0, 5)
            .forEach(System.out::println); // 0 -> 4
    IntStream.rangeClosed(0, 5)
                .forEach(System.out::println); // 0 -> 5
}
