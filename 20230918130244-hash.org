#+title: Hash

Mathematical function that converts a numerical input value into another compressed numerical value.
The input to the hash function is of *arbitrary* length but output is always of *fixed* length.

The output usually called a =hash= or =digest=

Can also be used to identify duplicate data sets in databases or archives to speed up searching of tables or to remove duplicate data.

Should not allow for: [[id:8c70f69c-ba76-42bf-8776-62612077dc4a][Hash Collisions]]
