#+title: Ciphertext-Only Analysis (COA)

Requires access to *one or more encrypted* messages.

No information is needed about the plaintext data, the [[id:d5db0593-2283-4d73-b904-bcd9f32717ce][algorithm]], or data about the cryptographic key.

Intelligence agencies face this challenge when intercepting encrypted communications with *no* key.
