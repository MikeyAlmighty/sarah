#+title: Cross-site scripting (XSS)

XSS attacks are a type of [[id:437f3cb0-292e-4224-b95a-bab932567176][Injection Attack]] where the attacker can insert malicious code and target the user of the service.

XSS attacks are a common method to achieve a [[id:7001dc45-f669-440c-bd54-38c5d5f21d8d][Session Hijacking/Cookie Hijacking]].
