1. Programming and Program Design F91PD

1.1. Learning Objectives

  1. Apply best practices in writing and running code for emerging industry problems.
  2. Use a modern programming language to design, implement, and test solutions.
  3. Apply procedural and OOP principles to produce well-designed, adaptable, and sustainable code.
  4. Analyse the applicability of data structures and code design choices to relevant problems and their implications in terms of efficiency of solution.
  5. Assess the use of software tools and languages in the design and development of solutions.
