#+title: Integer Data Types
Comprise the following primitive data types
- ~int~
- ~long~
- ~short~
- ~byte~

The default data type of an integer literal is always ~int~, but it can be specified as ~long~ by
appending the suffix L (or l) to the integer value. i.e. ~2023L~

There is no direct way to specify a ~short~ or a ~byte~ literal.
