* Assignment Operators
#+property: header-args :tangle code/ocp/2/assignment_operators.java

[[id:9bcf2aef-060a-4011-98b5-cb949a9f7d1a][Binary Operator]] that *modifies*, or *assigns*, the variable on the LHS of the operator with the result of the value on the RHS.
Unlike most other Java operators, the assignment operator is evaluated from ~right to left~.
