* Do While Loops

Unlike a [[id:9418ae56-d4dd-46e3-b7da-c404bc567e5e][while Loop]], though, a ~do/while~ loop guarantees that the statement or block will be executed *at least once*.

Use a *do-while* loop if you have statements inside the loop that must be *executed* *at* *least* *once*.

** Syntax

#+BEGIN_SRC java
do {
    // Loop body;
    Statement(s);
} while (loop-continuation-condition);
#+END_SRC

*** Example

#+BEGIN_SRC java
public class DoWhileLoop {
    public static void main(String[] args) {
        int count = 0;
        do {
            System.out.println("Welcome to Java!");
            count++;
        } while (count < 100);
    }
}
#+END_SRC
