#+title: Data Packet

=Data packet= is an all-encompassing term that represents any single set of binary data being sent across a network link.

The term data packet isn't tied to any specific layer or technology, it just represents a concept, one set of data being sent from point A to point B.

Data packets at the [[id:514c3f0b-c3ec-46f1-aa28-c2a4874a6a7b][Ethernet]] level are known as [[id:73b06fc5-e631-40ad-828f-741dd84bd15c][Ethernet Frames]].
