* Final Classes

The class may *not* be extended

#+BEGIN_SRC java
public final class Rhinoceros extends Mammal { }

public class Clara extends Rhinoceros { } // DOES NOT COMPILE
#+END_SRC


** _Exam Trick_

On the exam, pay attention to any class marked final.
If you see another class extending it, you know immediately the code does not compile.
