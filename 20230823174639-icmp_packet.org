#+title: ICMP Packet

 It has a header with a few fields
* Header
** Type Field
*8 bits*
Specifies what type of message is being delivered (Destination not reachable)

** Code Field
*8 bits*
Indicates specific code for error reason (Port unreachable, Destination unreachable etc.)

** Checksum Field
*16 bits*

** Rest of Header
*32 bits*
Optionally used by specific types/codes to send more data.

* Payload
Exists entirely so that the recipient of the message knows which of their transmissions caused the error being reported
It contains the entire =IP header= and the first *8 bytes* of the =data payload section= of the offending packet.
