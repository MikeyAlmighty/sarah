#+title: Networking

Computers on a network have an identifier called an [[id:1011a8fb-ad00-40bb-a1cc-4b53c9fedff0][IP Address]].

Devices that can connect to a network have another unique identifier called a [[id:5b075cfc-c900-4457-8811-27152afd5245][MAC Address]].

When you send or receive data through a network, you need to have both an [[id:1011a8fb-ad00-40bb-a1cc-4b53c9fedff0][IP Address]] and a MAC address.

* Casting
At the [[id:958904bc-6a62-45e5-a508-9f996ca87212][Network Layer]], looking at a =special bit= in the [[id:5b075cfc-c900-4457-8811-27152afd5245][MAC Address (MAC)]].

** 1. [[id:a983d5a3-cdfc-4c8c-98cb-a5656259506a][Unicast]]
If the =least significant bit= in the first Octet of a destination address is set to *0*,
It means that [[id:73b06fc5-e631-40ad-828f-741dd84bd15c][Ethernet Frame]] is intended for *only the destination address*.

** 2. [[id:c4309265-ff27-4592-85f6-3ed4ea47fc18][Multicast]]
If the =least significant bit= in the first Octet of a destination address is set to *1*,
It means that [[id:73b06fc5-e631-40ad-828f-741dd84bd15c][Ethernet Frame]] is intended for *a set of destination addresses*.
