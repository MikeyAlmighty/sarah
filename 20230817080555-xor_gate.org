#+title: XOR Gate

The XOR gate also involves two input signals.

The XOR rule outputs an =ON= signal when *only* one (but *not* both) of the inputs are =ON=.
Otherwise, the output signal will be =OFF=.

| INPUT A | INPUT B | OUTPUT |
|---------+---------+--------|
| ON      | OFF     | ON     |
| ON      | ON      | OFF    |
| OFF     | ON      | ON     |
| OFF     | OFF     | OFF    |
