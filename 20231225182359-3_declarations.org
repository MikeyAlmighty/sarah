#+title: 3. Declarations

This chapter covers declaration of ~classes~, ~methods~, ~constructors~, and ~variables~, including ~array types~.

* Exam Objectives
- Create ~classes~ and ~records~, and define and use ~instance~ and  [[id:fc670fe7-04be-4783-92b9-d97c7ee72bd7][Static Members]], [[id:73a0f36b-e440-486a-bd88-6ca577be3351][Constructors]], and ~instance~ and ~static~ initializers
- Implement *overloading*, including *var-arg* methods
- Understand variable ~scopes~, use local variable ~type inference~, apply *encapsulation*, and make objects *immutable*
- Create Java ~arrays~, ~List~, ~Set~, ~Map~ and ~Deque~ *collections* and add, remove, update, retrieve and sort their elements.
