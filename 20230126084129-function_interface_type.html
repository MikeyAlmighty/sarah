Interface Function <T, R>

T = Type of Argument R = Type of Result

There are special flavors: BiFunction (Takes in two arguments and returns a result) Interface BiFunction<T, U, R>

1. Primitives:

  1. Return Type is paramterized

IntFunction (Takes in primitive int) Also have LongFunction, DoubleFunction etc.

  1. Argument Type is paramterized

ToIntFunction (Function that returns primitive integer)

  1. Both Argument and Return type are paramterized

IntToLongFunction 

public static void show() {
    // Takes a String and returns int
    Function<String, Integer> map = str -> str.length();

    var stringLength = map.apply("Sky");
    System.out.println(stringLength);
}

2. Function Compostion

public static void show() {
    // Have "key:value"
    // first: "key=value"
    // second: "{key=value}"

    // First
    Function<String, String> replaceColon = str -> str.replace(":", "=");
    // Second
    Function<String, String> addBraces = str -> "{" + str + "}";

    var result = replaceColon
                    .andThen(addBraces)
                    .apply("key:value");

    System.out.println(result); // "{key=value}"

    // Alternatively using `compose`:
    var result1 = addBraces
                    .compose(replaceColon)
                    .apply("key:value");


    System.out.println(result1); // "{key=value}"
}

3. BinaryOperator

public static void show() {

    BinaryOperator<Integer> add = (num1, num2) -> num1 + num2;
    Function<Integer, Integer> square = num1 -> num1 * num1;

    var result = add
                    .andThen(square)
                    .apply(1, 2);
    System.out.println(result); // 9
}

4. UnaryOperator

public static void show() {
    UnaryOperator<Integer> square = n -> n * n;
    UnaryOperator<Integer> increment = n -> n + 1;

    var result = increment.andThen(square).apply(1);
    System.out.println(result);
}
