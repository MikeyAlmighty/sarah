1. Shortening an IPv6 address:

  1. Remove any leading zeros from a group.
  2. Any number of consecutive groups composed of just zeros can be replaced with two colons ::.

2. Anatomy

NetworkID: First 64 bits HostID: Second 64 bits

3. Reserved Address Spaces

Documentation: Start with 2001:0db8 If it starts with FF00:xxx Multicast If it starts with FE80:xxx Link-Local Unicast

4. Header

4.1. Version Field

4 bit Defines what version of IP is in use

4.2. Traffic Class Field

8 bit Defines type of traffic contained within datagram

4.3. Flow Label Field

20 bit Used in conjunction with Traffic Class field for routers to make decisions about the quality of service level for a specific datagram.

4.4. Payload Length Field

16 bit Defines how long data payload section of datagram is.

4.5. Next Header Field

16 bit Optional

4.6. Hop Limit Field

8 bit Identical to TTL Field in IPv4 Header

4.7. Source Field

128 bit

4.8. Destination Field

128 bit
