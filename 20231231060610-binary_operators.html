1. Binary Operator(s)

Operators that take two operands

A unary Operator has only one operand (binary operator has two).

For example, the – operator in –5 is a unary operator to negate number 5, whereas the – operator in 4 – 5 is a binary operator for subtracting 5 from 4.

Used to:

  • Perform mathematical operations on variables
  • Create logical expressions
  • Perform basic variable assignments.
