#+title: Three Way Handshake


Computer A -> =ACK= -> Computer B
Computer A <- =SYN/ACK= <- Computer B
Computer A -> =ACK= -> Computer B

Since both sides have now sent =SYN/ACK= pairs to each other, a [[id:690db04e-097b-4e6c-8e78-92e353fc49da][TCP]] connection in this state is operating in [[id:ff8ce9c1-ff52-4f5d-aad3-21f0d144d76d][Full Duplex]].
