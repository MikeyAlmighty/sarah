#+title: DHCP Discovery Process

* 1. The DHCP client sends a [[id:66227676-22f8-4c87-b1d2-3e13119ba934][DHCPDISCOVER]] message out onto the network.
* 2. DHCP responds with [[id:848cf669-5745-47e3-bb6f-e8cb08ab7619][DHCPOFFER]]
* 3. DHCP Client will respond with [[id:597ff77f-36b7-435c-aa33-7fcec0e77124][DHCPREQUEST]] message
* 4. DHCP Server sends respond with [[id:039a7c82-da04-4ba6-a749-c6108c0bcad5][DHCPACK]] message
