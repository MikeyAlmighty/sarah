* Parameter List

** Empty List

Although the parameter list is *required*, it doesnâ€™t *have to contain any parameters*.
This means you can just have an empty pair of parentheses after the [[id:0b8acbe2-0b2b-4829-8aab-804dde49cabf][Method Name]]:

*** Example

#+BEGIN_SRC java
public class Sleep {
    void nap() {}
}
#+END_SRC

** Multiple paramaters

If you do have *multiple* parameters, you separate them with a =comma=.

*** Example

#+BEGIN_SRC java
public class PhysicalEducation {
    public void run1() {}
    public void run2 {}
    public void run3(int a) {}
    public void run4(int a; int b) {}
    public void run5(int a, int b) {}
}
#+END_SRC
