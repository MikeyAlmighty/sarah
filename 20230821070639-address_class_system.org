#+title: Address Class System

Defines how global [[id:1011a8fb-ad00-40bb-a1cc-4b53c9fedff0][IP Address]] space is split up

(This classing system is mostly replaced by [[id:dedd1989-010f-480e-8ab2-8f64dcf3aaca][CIDR]]).

There are three primary types of address classes:

* Class A
Range: 0 - 127
Max Hosts: 16 million
First Octets: Network ID
Last three Octets: Host ID

* Class B
Range: 128 - 191
Max Hosts: 64 000
First two octets: Network ID
Second two Octets: Host ID

* Class C
Range: 192 - 223
Max Hosts: 254
First three Octets: Network ID
Last Octet: Host ID
