* .random()

The ~random()~ method returns a value *greater than* or *equal* to =0= and *less* than =1=.
Generates a *random* ~double~ value greater than or equal to =0.0= and less than =1.0= (~0 <= Math.random() < 1.0~).

** Signature

#+BEGIN_SRC java
public static double random()
#+END_SRC

*** Example

#+BEGIN_SRC java
double num = Math.random();
#+END_SRC

#+BEGIN_SRC java
// 0 to 9
(int)(Math.random() * 10);

// 1 to 10
(int)(Math.random() * 10) + 1;

// min to max (inclusive)
int n = (int)(Math.random() * (max - min + 1)) + min;
#+END_SRC

*** Note
For more control, consider using ~java.util.Random~.
