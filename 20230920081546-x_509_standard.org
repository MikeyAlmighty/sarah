#+title: X.509 Standard

First issued in 1988

Defines the format of [[id:dc321a18-5fdd-49c8-8969-0a4752dcb1dd][Digital Certificates]]
Also defines a [[id:667392b4-e509-4543-8491-3b322de3113b][Certificate Revocation List (CRL)]]

* Defined Fields:
- *Version:* (Modern version: =3.0=)
- *Serial Number:* (ID for the certificate assigned)
- *Certificate Signature Algorithm:* (*Public key alogrithm* is used for the =Public key= and what [[id:48e1ad8d-51d5-4656-b292-8f2ba311f061][hashing]] algorithm is used for =signing=)
- *Issuer Name:* (=Authority= that signed the certificate)
- *Validity:* (Contains two subfields which defines the dates certificate is valid: 1. =Not Before= 2. =Not After=)
- *Subject:* (Identifying information about the entity the certificate was issued to)
- *Subject Public Key Info:* (Define the =algorithm= of the public-key along with the =public-key= itself)
- *Certificate Signature Value:* (=Digital Signature= data itself)
