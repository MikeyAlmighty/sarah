#+title: OpenLDAP

Free and Open Source
Supports [[id:45bb23df-e29d-46dc-b704-f17cf26af1d7][Windows]] and [[id:a5441999-e7a7-4034-b996-cfc8462989ed][Linux/Unix]]

* Commands
=ldapadd= Takes input of LDIF file and adds context of files
=ldapmodify= Modifies an existing object
=ldapdelete= Remove object that the LDIF file refers to
=ldapsearch= Search for entries in your directory DB
