* Programmable Data Processors

The [[id:215906d9-9200-42b8-afbf-3c30b0f6c585][Turing Model]] is a better model for a ~general-purpose computer~ than a ([[id:c7be4c6a-69ce-4d2e-b6af-2f8d285e1160][Single puprose computing machine]]).

This model adds an extra element to the specific computing machine: the [[id:899ca310-9704-46b5-9177-b2d76a875456][program]].

#+ATTR_ORG: :width 500
[[./assets/turing_model_programmable_data_processor.png]]

In the [[id:215906d9-9200-42b8-afbf-3c30b0f6c585][Turing Model]], the ~output data~ depends on the combination of *two* factors:
- Input data
- The [[id:899ca310-9704-46b5-9177-b2d76a875456][program]]

** Processing

- With the same ~input data~, we can generate *different* output if we change the [[id:899ca310-9704-46b5-9177-b2d76a875456][program]].
- With the same [[id:899ca310-9704-46b5-9177-b2d76a875456][program]], we can generate different ~outputs~ if we change the ~input data~.
