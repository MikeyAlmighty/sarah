* Creating an Array of Reference Variables

You can choose any Java ~type~ to be the type of the array.

#+BEGIN_SRC java
String[] bugs = { "cricket", "beetle", "ladybug" };
String[] alias = bugs;

System.out.println(bugs.equals(alias)); // true
System.out.println(bugs.toString()); // [Ljava.lang.String;@160bc7c0
#+END_SRC
We can call ~equals()~ because an array is an object.

It returns true because of *reference equality*.

The ~equals()~ method on arrays does *not* look at the elements of the array.

** _Exam Trick_

Remember, this would work even on an ~int[]~ too. The type ~int~ is a [[id:46582863-7f97-448d-bbc7-f7da4c1e7574][primitive]]; ~int[]~ however is an object.



#+BEGIN_SRC java
int[] nums = { 2, 45, 1, 3 };
int[] nums2 = nums;

System.out.println(nums.equals(nums2)); // true

int[] nums = { 2, 45, 1, 3 };
int[] nums2 = { 2, 45, 1, 3 };
System.out.println(nums.equals(nums2)); // false
#+END_SRC
