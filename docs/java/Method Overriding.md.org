* Method Overriding
:PROPERTIES:
:CUSTOM_ID: method-overriding
:END:
Override (Use =@Override=) any method we inherit with a different/our
own implementation.

#+begin_src java
   @Override // Override toString in parent Class
   public String toString(){
      var overriddenText = "Overridden mfer";
      return overriddenText;
   }
#+end_src
