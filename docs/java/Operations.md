# Operations

```java
int result = 10 + 3;
int result1 = 10 * 3;
double result2 = (double)10 / (double)3; // Cast
System.out.println(result2);
```
# Postfix
```java
int x = 1;
int y = x++; // Postfix (copied to Y then x is incremented)
System.out.println(y); // 1
System.out.println(x); // 2
```
```java
int x = 1;
int y1 = ++x; // Prefix (x is incremented then copied to y)
System.out.println(y1); // 2
System.out.println(x); // 2
```
