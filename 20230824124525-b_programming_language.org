#+title: B Programming Language

B is a programming language developed at [[id:55e2923f-2df5-4586-843e-4abff57bad32][Bell Labs]] circa 1969 by [[id:2f862a56-fd11-431b-b8d1-812aafcacd7f][Ken Thompson]] and [[id:7994c754-f128-411a-94b2-11b59350efa0][Dennis Ritchie]].
