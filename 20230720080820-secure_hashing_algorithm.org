#+title: Secure Hashing Algorithm (SHA)

* SHA-1
=SHA-1= Developed and released by NSA (1995)
=SHA-1= takes any length of string and outputs 160bit value.
=SHA-1= is a cryptographic hash function, but is *NOT* an encryption function.

Operates on *512-bit* blocks
Generates: *160-bit* Hash/Digest
