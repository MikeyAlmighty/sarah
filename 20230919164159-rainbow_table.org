#+title: Rainbow Table

A precomputed table for caching the outputs of a cryptographic [[id:48e1ad8d-51d5-4656-b292-8f2ba311f061][Hash]] function, usually for cracking password hashes.

Passwords are typically stored not in plain text form, but as hash values.
