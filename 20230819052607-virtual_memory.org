#+title: Virtual Memory

The combination of [[id:7a9a7651-d42f-4cd9-a140-c04c3704e2db][Hard Drive]] space and [[id:73ee7657-201d-4041-acfd-c05bdfaeb118][RAM]] that acts like =memory= that our [[id:2bc74ef5-d9b1-4286-ab74-bc061c6a1704][processes]] can use.

When we execute a [[id:2bc74ef5-d9b1-4286-ab74-bc061c6a1704][process]], we take the data of the program in chunks we call *pages*.

We store these *pages* in =Virtual Memory=.
