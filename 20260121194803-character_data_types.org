#+title: Character Data Types

* Character Data Types
A =character= *data* *type* represents a ~single character~.
In addition to processing numeric values, you can process characters in Java.
The =character= *data* *type*, ~char~, is used to represent a ~single character~.

** char
Size: =16= =bits=
Range: =0= to =65,535= (represents a single Unicode character)
Use case: Used for storing *characters*, such as ~letters~, ~digits~, or ~symbols~.

*** Example

#+BEGIN_SRC java
char letter = 'A';
char numChar = '4';
#+END_SRC
