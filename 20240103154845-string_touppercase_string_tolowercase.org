* String.toUpperCase() and String.toLowerCase()

- ~toUpperCase()~ method converts any *lowercase* characters to *uppercase* in the returned string.
- ~toLowerCase()~ method converts any *uppercase* characters to *lowercase* in the returned string.
These methods *leave alone* any characters other than *letters*.

*** Signature

#+BEGIN_SRC java
public String toLowerCase()
public String toUpperCase()
#+END_SRC

**** Example

#+BEGIN_SRC java
var name = "animals";

System.out.println(name.toUpperCase()); // ANIMALS
System.out.println("Abc123".toLowerCase()); // abc123
#+END_SRC
