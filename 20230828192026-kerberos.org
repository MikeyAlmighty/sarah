#+title: Kerberos

Network =Authentication= Protocol Used to:
- Authenticate user identity
- Secure the transfer of user credentials and more.


Uses *tickets* to allow entities to prove their identity over potentially insecure channels to provide mutual authentication
Also uses [[id:df2a4b55-348c-4450-91ec-209c7060c154][Symmetric Encryption]] to *protect* protocol messages from =eavesdropping= and =replay attacks=
