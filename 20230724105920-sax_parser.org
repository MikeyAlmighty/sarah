#+title: SAX parser

* Simple API for XML (SAX)
SAX is an event-driven online algorithm for =lexing= and =parsing= XML documents, with an API developed by the XML-DEV mailing list.
SAX provides a mechanism for reading data from an XML document that is an *alternative* to that provided by the DOM.
