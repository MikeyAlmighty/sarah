#+title: Math.max
#+property: header-args :tangle code/ocp/4/math_max.java

* .max()

Compare two values and return *largest* one of them.

** Signature

#+BEGIN_SRC java
public static double max(double a, double b)
public static float max(float a, float b)
public static int max(int a, int b)
public static long max(long a, long b)
#+END_SRC

*** Example

#+BEGIN_SRC java
int first = Math.max(3, 7);  // 7
#+END_SRC
