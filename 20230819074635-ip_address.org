#+title: IP Address

An [[id:9bf3b633-f396-4e1b-9b55-6c6b06d60bec][IP]] address is composed of digits and numbers like =100.1.4.3=

* IPV4 Addresss
32 bits separated into four groups (=172.4.3.1=)

* IPV6 Addresss
128 bits separated into four groups (=2001:db8:3333:4444:5555:6666:7777:8888=)
