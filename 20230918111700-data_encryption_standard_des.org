#+title: Data Encryption Standard (DES)

Designed in the 1970s by [[id:f45c15e5-a25a-4f90-9f28-b97c103f81bf][IBM]] with some input form the *NSA*
Adopted as a [[id:e763f423-898d-462d-b23e-1457cc6b7180][FIPS]] for the *US*

[[id:735e47da-fa00-497b-b707-ba607c1ba7ed][Symmetric]] [[id:2acddcbe-0490-4b4b-9c80-707dc3592d9c][Block Cipher]] that uses *64-bit* key sizes and operates on blocks *64-bits* in size

Though the key size is technically *64-bits* in length, *8-bits* are used only for [[id:1632d1ee-6f1f-4a9c-85fd-bd8914e1d1ce][Parity checking]]

Key Length = *56-bits* (72 quadrillion possible keys)

Brute forced: [[id:c9cff436-2cb5-413d-a2e8-b9ef7d72daad][Electronic Frontier Foundation (EFF)]]

Replaced by [[id:03e91c13-0982-4f85-9e75-f7664a6359f9][AES]]
