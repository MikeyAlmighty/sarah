#+title: DHCPOFFER

Destination port of =:68=,
Source port of =:67=
Destination broadcast IP of =255.255.255.255=
Source: Actual IP

(Since the =DHCPOFFER= is also a [[id:07f12bda-2f73-4fe3-9570-6734585201e6][broadcast]], it would reach *every* machine on the network.
The original client would recognize that this message was intended for itself.
This is because the =DHCPOFFER= has the field that specifies the [[id:5b075cfc-c900-4457-8811-27152afd5245][MAC Address]] of the client that sent the =DHCPDISCOVER= message.)
