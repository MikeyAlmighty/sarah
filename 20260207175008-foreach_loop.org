#+title: Foreach Loop

* Foreach Loop

Enables you to *traverse* the =array= *sequentially* without using an *index* *variable*.

** Syntax

#+BEGIN_SRC java
for (elementType element: arrayRefVar) {
    // Process the element
}
#+END_SRC

*** Example

#+BEGIN_SRC java
for (double e: myList) {
    System.out.println(e);
}
#+END_SRC

** Exam Tip
Accessing an array out of bounds is a common programming error that throws a runtime ~ArrayIndexOutOfBoundsException~.

To avoid it, make sure you do not use an *index* beyond ~arrayRefVar.length âˆ’ 1~ or simply using a *foreach* loop if possible.

Programmers often mistakenly reference the *first* element in an array with index =1=, but it should be =0=.

This is called the *off-by-one error*.
Another common *off-by-one* error in a loop is using ~<=~ where ~<~ should be used.
