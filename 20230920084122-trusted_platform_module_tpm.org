#+title: Trusted Platform Module (TPM)

This is a *hardware device* that's typically integrated into the hardware of a computer that's a dedicated crypto processor.

Most commonly used to:
- Ensure platform integrity
- Preventing unauthorized changes to the system either in software hardware
- [[id:f0531aa9-40ac-48a1-8603-643e4e9b019c][Full Disk Encryption (FDE)]] (utilizing the TPM to protect the entire contents of the disc)

TPMs offer:
- secure generation of keys
- Random number generation
- [[id:2a174ca3-e2ca-4a3b-adab-6b6a793d147c][Remote attestation]]
- Data binding and seiling


Mobile devices have something similar: [[id:6a37be74-d658-4fd9-8781-fcb24ea63515][Secure Element]]
