#+title: Network Address Translation (NAT)

Lets organizations use one *public* [[id:1011a8fb-ad00-40bb-a1cc-4b53c9fedff0][IP Address]] and many *private* [[id:1011a8fb-ad00-40bb-a1cc-4b53c9fedff0][IP Addresses]] within the network.

=NAT= is a technology that allows a gateway, (router/firewall) to rewrite the source [[id:9bf3b633-f396-4e1b-9b55-6c6b06d60bec][IP]] of an outgoing [[id:6fcc1f55-523e-4e97-9311-2ee521d4d7fd][IP Datagram]] while retaining the original IP in order to rewrite it into the response.
