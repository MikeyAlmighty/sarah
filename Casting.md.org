* Implicit Casting

Also known as ~automatic type conversion~, *widening* happens when a *smaller* data type is automatically converted into a *larger* data type.

Implicit (*automatically*) casting (No precision loss).

#+begin_src java
// byte > short > int > long > float > double
public class WideningExample {
        public static void main(String[] args) {
             int num = 100;
             double doubleNum = num;// Automatic conversion from int to double
             System.out.println("Integer value: " + num);
             System.out.println("Converted to double: " + doubleNum);
        }
}

#+end_src
