* Static Variable Modifiers

=Static= variables can be declared with the same [[id:ce19a18e-7f0c-475c-a05b-facbf29af10d][instance Variables]], such as ~final~, ~transient~, and ~volatile~.
=Static= variables that are never meant to change are known as a *constant*.
It uses the ~final~ modifier to ensure the variable *never changes*.

*Constants* use the modifier ~static final~ and a different naming convention than other variables.
They use *all uppercase letters* with *underscores between words*.

** Example

The compiler will make sure that you do not accidentally try to update a ~final~ variable.

#+BEGIN_SRC java
public class ZooPen {
    private static final int NUM_BUCKETS = 45;
    public static void main(String[] args) {
        NUM_BUCKETS = 5; // DOES NOT COMPILE
    }
}
#+END_SRC
