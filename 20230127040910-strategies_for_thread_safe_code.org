#+title: Strategies for Thread safe code

* [[id:0a07a8a7-9849-46d6-8b8b-22e187f71080][Confinement]]
Don't share data across threads

* Immutable
Use immutable objects

* [[id:e69555ef-ade4-4b3e-ae63-59d3d6182b33][Synchronization]]
Prevent multiple threads from accessing same object concurrently

* [[id:4fbdf6dd-f4c9-4636-b7ff-d61bdd030ba5][Atomic Objects]]
Increment atomicIntegers
(Atomic operations)

* Partitioning
Partitioning data into segments that can be access concurrently.
