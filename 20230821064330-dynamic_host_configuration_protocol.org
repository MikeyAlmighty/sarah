#+title: Dynamic Host Configuration Protocol (DHCP)

[[id:8580f7d1-7d1b-4922-9e73-7a7e2c9318bd][Application Layer]] protocol that *automates* the configuration process of =hosts on a network=.

Every single computer on a modern TCP IP-based network needs to have:
1. [[id:1011a8fb-ad00-40bb-a1cc-4b53c9fedff0][IP Address]]
2. [[id:94d758d7-4778-4319-ad8d-17c9f0859ec9][Subnet Masks]]
3. Primary gateway
4. Name Server.
