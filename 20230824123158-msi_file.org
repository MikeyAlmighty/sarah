#+title: .Microsoft Install package (.msi)

A =Microsoft install package= or =MSI= is used to guide a program called the Windows installer for:
- Installation
- Maintenance
- Removal of programs on the [[id:45bb23df-e29d-46dc-b704-f17cf26af1d7][Windows]] [[id:1dcff204-74cf-445d-9357-5c01ca3d2a03][OS]].
