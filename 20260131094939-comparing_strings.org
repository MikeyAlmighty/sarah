#+title: Comparing Strings

* Comparing Strings

| Method                  | Description                                                                     | Example                                  |
|-------------------------+---------------------------------------------------------------------------------+------------------------------------------|
| equals(s1)              | Returns true if this string is equal to string s1.                              | "Hello".equals("Hello") → true           |
| equalsIgnoreCase(s1)    | Returns true if this string is equal to string s1; case insensitive.            | "Hello".equalsIgnoreCase("hello") → true |
| compareTo(s1)           | Returns >0, 0, or <0 if this string is greater than, equal to, or less than s1. | "cat".compareTo("car") → >0              |
| compareToIgnoreCase(s1) | Same as compareTo, but case insensitive.                                        | "Apple".compareToIgnoreCase("apple") → 0 |
| startsWith(prefix)      | Returns true if this string starts with the specified prefix.                   | "programming".startsWith("pro") → true   |
| endsWith(suffix)        | Returns true if this string ends with the specified suffix.                     | "filename.txt".endsWith(".txt") → true   |
| contains(s1)            | Returns true if s1 is a substring of this string.                               | "hello world".contains("world") → true   |

** Note
*Syntax* *errors* will occur if you compare strings by using [[id:bf93d4e3-f501-4e73-ac07-e1c6653831ba][relational operators]] ~>, >=, <, or <=~.
Instead, you have to use ~s1.compareTo(s2)~.

- The ~equals~ method returns =true= if two strings are *equal*, and =false= if they are *not*.
- The ~compareTo~ method returns =0=, a positive integer, or a negative integer, depending on whether one string is equal to, greater than, or less than the other string.
