* Method Body

A ~method body~ is simply a *code block*. It has braces that contain *zero* or *more* Java statements

** Example

#+BEGIN_SRC java
public class Bird {
    public void fly1() {}
    public void fly2() // DOES NOT COMPILE
    public void fly3(int a) { int name = 5; }
}
#+END_SRC
