#+title: SQL Injection

Targets the entire website if the website is using a [[id:e86502bc-990e-4ac3-86e9-4b06f475b5e9][SQL]] database.

Attackers can potentially run SQL commands that allow them to delete website data, copy it and run other malicious commands.
